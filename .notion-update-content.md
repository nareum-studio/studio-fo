# 나름다움 스튜디오 (nareum-studio) 프로젝트

사진 스튜디오 웹사이트 프로젝트입니다. "나만의 아름다움을 담아드리는 나름다움"을 콘셉트로 한 NareumDaumm Studio 웹사이트를 개발 중입니다.

---

## 프로젝트 개요

- **프로젝트명**: nareum-studio
- **콘셉트**: 나만의 아름다움을 담아드리는 나름다움
- **서비스**: 배우프로필 | 키즈프로필 | 소장용프로필 | 커플사진

## 페이지 구조

### 사용자 페이지 (users)
- **ABOUT** (/) - 메인 소개
- **PORTFOLIO**
  - PROFILE (/portfolio/profile)
  - KIDS (/portfolio/kids)
  - BALLET (/portfolio/ballet)
- **ARTISTIC** (/artistic)
- **PRICE** (/price)
- **CONTACT** (/contact)

### 관리자 페이지 (admin)
- **로그인** (/admin/login)
- **관리자** (/admin) - 소개글 수정, 등록된 사진 수정 (GalleryManager)

---

## 주요 기능

1. **소개글 수정**: IntroEditor로 스튜디오 소개 텍스트 편집
2. **갤러리 관리**: profile/kids/ballet 섹션별 사진 업로드, 삭제 (GalleryManager)
3. **반응형 네비게이션**: Header, MobileMenu
4. **포트폴리오 갤러리**: 프로필, 키즈, 발레 카테고리별 포트폴리오 표시

---

## 폴더 구조

- app/(users)/ - 사용자 페이지 (라우트 그룹)
- app/admin/ - 관리자 페이지
- components/ - UI 컴포넌트, header, admin
- public/data/ - nav 설정, 타입 정의
